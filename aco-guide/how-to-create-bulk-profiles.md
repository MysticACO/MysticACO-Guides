# Submitting Bulk

## Step-by-step guide

* When running multiple entries for ACO checkouts, each profile must appear unique to the retailer’s system. Retailers often cancel orders when duplicate information is detected, so it’s important that every entry uses **distinct account details**.

> A typical bulk setup requires four main elements: **unique cards, emails, addresses, and phone numbers.**
