# IMAP Tutorial

IMAP lets us read verification emails directly from your mail provider.  
This is required for sites that need verification codes or password resets.

<div class="alert danger">

## ‚ö† REQUIRED FOR CHECKOUT
If IMAP is not configured correctly, we will NOT receive your verification codes.

This can cause failed checkouts and missed items.  
Mystic ACO is not responsible for missed purchases due to incorrect email setup.

</div>

---

<div class="card">

## What IMAP Allows Us To Do
- Automatically read verification codes (Nike, SNKRS, Shopify, Target, Walmart)
- Confirm account logins
- Handle password resets

### What We CANNOT See
- We cannot send emails
- We cannot access personal messages
- We only access retailer verification emails

## Gmail Setup
1. Turn on 2-Step Verification  
https://myaccount.google.com/signinoptions/two-step-verification/enroll-welcome

2. Create App Password  
https://myaccount.google.com/apppasswords

3. Submit the 16-character app password.

---

## Yahoo Setup
1. https://login.yahoo.com/myaccount/security  
2. Scroll to External Connections  
3. Create App Password

---

## iCloud Setup
1. https://appleid.apple.com  
2. Enable Two-Factor Authentication  
3. Security ‚Üí App-Specific Passwords ‚Üí Generate

---

## AOL Setup
1. https://login.aol.com/myaccount/security/  
2. Generate App Password
